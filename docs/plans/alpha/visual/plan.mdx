## Outcome

The visual shows the complete intended creator journey while keeping the pre-freeze implementation boundary narrow. The critical P0 sequence is recommendation → accepted starter-aware roadmap → bounded generated quest → independent proof → creator play/confirmation → atomic completion/restart. Idea promotion, Companion conversation, and Web export are visible as future states so their boundaries can be reviewed, not as implementation commitments.

<Callout id="scope-decision" tone="decision">

**Implement only the P0 spine before freeze.** Existing v0.2 surfaces are reused. Open in Godot is an optional P1 action. Export readiness and showcase refresh are P2 submission work. Idea refinement, proposal promotion, and Companion conversation remain deferred.

</Callout>
